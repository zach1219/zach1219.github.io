## 关于

站点名称：好好吃饭
作者：zach
地址：[https://zach1219.github.io](https://zach1219.github.io)

## 内容

- **作品** — 个人项目展示
- **简历** — 个人简介

## 技术栈

- Jekyll + GitHub Pages
- Academic Pages 模板
- Utterances 评论系统（基于 GitHub Issues）

## 本地开发

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```
